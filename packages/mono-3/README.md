# @phathdt/mono-3

Additional features package for mono-sdk. Depends on `@phathdt/mono-1`.

## Installation

```bash
# npm
npm install @phathdt/mono-3

# yarn
yarn add @phathdt/mono-3

# pnpm
pnpm add @phathdt/mono-3
```

## Usage

```typescript
import { hello4 } from '@phathdt/mono-3';

hello4(); // logs greeting with hello1 result
```

## API

### `hello4(): void`

Logs a greeting message using `hello1()` from mono-1.

## Dependencies

- `@phathdt/mono-1` - Automatically installed as a dependency
