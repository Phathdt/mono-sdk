# @phathdt/mono-2

Extended features package for mono-sdk. Depends on `@phathdt/mono-1`.

## Installation

```bash
# npm
npm install @phathdt/mono-2

# yarn
yarn add @phathdt/mono-2

# pnpm
pnpm add @phathdt/mono-2
```

## Usage

```typescript
import { hello2, hello3 } from '@phathdt/mono-2';

hello2(); // logs greeting with hello1 result
hello3(); // logs greeting with hello1 result
```

## API

### `hello2(): void`

Logs a greeting message using `hello1()` from mono-1.

### `hello3(): void`

Logs a greeting message using `hello1()` from mono-1.

## Dependencies

- `@phathdt/mono-1` - Automatically installed as a dependency
