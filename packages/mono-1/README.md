# @phathdt/mono-1

Base utilities package for mono-sdk.

## Installation

```bash
# npm
npm install @phathdt/mono-1

# yarn
yarn add @phathdt/mono-1

# pnpm
pnpm add @phathdt/mono-1
```

## Usage

```typescript
import { hello1 } from '@phathdt/mono-1';

const result = hello1(); // returns 789
```

## API

### `hello1(): number`

Returns a number value.
