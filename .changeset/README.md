# Changesets

This folder has been automatically generated by `@changesets/cli`.

## Creating a Changeset

1. Run `pnpm changeset` in the root directory
2. Select the packages that have changed
3. Choose the bump type (major, minor, patch)
4. Write a summary of your changes

## Releasing

Releases are automated via GitHub Actions:
1. When PRs with changesets merge to main, a Version PR is created
2. Merging the Version PR triggers the publish workflow
3. Packages are published to GitHub Packages npm registry
